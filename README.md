Desafio Técnico: Analista de Dados Júnior
Empresa: genCripto Área: Data & Analytics
Sobre a genCripto
A genCripto é uma wealthtech em rápida expansão, focada em democratizar o acesso a ativos tradicionais e criptomoedas. Nossa plataforma web já é um sucesso entre os clientes, mas nosso back-office está crescendo mais rápido do que conseguimos acompanhar.
Acreditamos que dados não servem apenas para contar o que aconteceu, mas para guiar nossas próximas decisões estratégicas. E é por isso que precisamos de você.
O Seu Desafio
Nosso time de Produto acabou de lançar a nova interface do cliente (Front-End), onde eles podem visualizar seus ativos. No entanto, o C-Level (Diretoria) precisa de uma visão executiva do risco e da rentabilidade da nossa carteira de clientes.
Hoje, os dados transacionais estão soltos em arquivos CSV extraídos do nosso sistema legado, e os dados reais de mercado não estão integrados.
Seu objetivo como nosso novo Analista de Dados será construir o pipeline completo dessa informação: desde a modelagem no banco de dados até a visualização final para a diretoria.
Etapas do Projeto
Fase 1: Fundações de Dados (SQL)
Recebemos um dump do sistema com dois arquivos: clientes.csv e transacoes.csv. Sua primeira missão é estruturar nosso banco de dados relacional.
Ação: Crie as tabelas (Dimensão e Fato) garantindo as tipagens corretas e as chaves (PK/FK).
Business Intelligence (Power BI)
Chegou a hora de apresentar seu trabalho para a Diretoria.
Ação: Importe a base do banco de dados (SQL) e conecte-a com a pasta contendo os dados de mercado extraídos pelo Python. Modele os dados corretamente.
Entrega: 
Um Dashboard interativo. Ele deve responder rapidamente:
Qual é o volume total custodiado pela corretora?
Qual cliente tem o maior capital investido?
Qual ativo da nossa base teve a maior valorização comparando o preço médio de compra com o preço de mercado atual?
